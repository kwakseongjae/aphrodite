# Editorial Portfolio — solo-maker register

## When this skill applies

Intent mentions: portfolio, work-grid, solo practitioner, craft/maker, furniture, architecture, ceramics, photographer, designer. NOT for SaaS, dashboards, marketing landing pages.

## Workflow

1. **Reference orientation.** Look at 3–5 examples of magazine-pacing portfolio sites (think Cereal, Apartamento, Kinfolk web translations — editorial, generous whitespace, photo-led).
2. **Palette guardrails.** Warm-neutral primary keyed to the maker's material (walnut → grey-brown ochre; ceramic → stoneware grey; etc.). Counter with cool-violet neutrals to avoid one-note warmth. Avoid pure blue accents unless the maker's brand calls for it.
3. **Type pairing.** Display should be a **contemporary serif** with optical sizing (Source Serif 4, Newsreader, Fraunces display optical). Avoid Renaissance / olde-style fares (EB Garamond, Cormorant) — wrong register for a working studio. Body sans is fine (Outfit, Inter at light weight).
4. **Spacing scale.** Push max token to **160–224 px**. Magazine pacing requires breathing room between projects; a 96 px cap will collapse the design's visual rhythm.
5. **Brand-a variant.** Should be **near-white gallery wall** (#faf9f6 territory). Avoid parchment-yellow casts that compete with material photography.
6. **Composition structure.** Hero + Selected Work grid (image-dominant cards, narrow text rail) + minimal About + plain-text contact (no form widgets — "a person you call, not a brand you submit tickets to").
7. **Validation.** WCAG-AA on all four variants. Prose-vs-tokens audit: any claim in the body ("generous spacing") must be reflected in the tokens.

## Do / Don't

- DO let photography dominate. DO use full-bleed images.
- DO use sharp corners (radius 2–6 px max — joinery, not soft-tech).
- DON'T add SaaS-style stats rows ("12 years / 87 pieces / 24 clients"). The work is the metric.
- DON'T use form widgets for contact. Plain text email + phone + address.
- DON'T pick blue as a primary unless the maker's actual brand demands it.
