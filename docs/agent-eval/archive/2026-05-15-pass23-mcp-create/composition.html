Cord Radio

A wall-mounted radio with a pull-cord power switch

A single gesture — reach, pull, listen. The radio mounts to the wall the way a light switch does, at shoulder height, where the hand finds it without looking. The pull cord is the only control. One pull brings the radio to life. A second pull silences it. There is no standby light, no display, no pairing procedure.

The housing is cast aluminum, 3.4 mm, finished in a single matte colour chosen to recede against any wall it is placed on. The speaker grille is stamped in a single pass — 312 perforations, each 2.1 mm, forming a pattern that reads as a uniform texture from a meter away and reveals its precision up close.

We did not design an object. We designed the moment the hand meets the cord — the weight of the pull, the click, the sound that follows.
